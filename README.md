# 🌍 World Monitor – Global Intelligence Dashboard

A modern and interactive **Global Intelligence Dashboard** built using **React.js**, designed to visualize worldwide geopolitical events, strategic insights, cybersecurity threats, and global intelligence in a single responsive interface.

The dashboard provides an intuitive experience for monitoring worldwide activities through an interactive world map, AI-powered analytics, live intelligence feeds, strategic risk indicators, and regional monitoring panels.

---

## 🚀 Features

### 🌎 Interactive World Map
- Interactive global map visualization
- Region-wise monitoring
- Dynamic markers and connection paths
- Zoom and navigation controls

### 📊 Global Monitoring
- Worldwide situation overview

- Regional intelligence tracking
- Global activity visualization

### 🤖 AI Intelligence
